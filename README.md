# VoiceChatPyroBot

## Deployment

- Configure
    - Passing CLI args when running the bot
    - Or copying config/sample_config.py to config/config.py
    
- Install the dependencies 
    ```shell
    sudo apt install mpv libmpv-dev pulseaudio &&
    pip3 install -U -r requirements.txt
    ```
- Run
    ```shell
  python3 main.py  
  ```
